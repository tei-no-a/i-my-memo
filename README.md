# I My Memo
あいまいメモは、「あいまいでもメモが書ける」がコンセプトの軽量なWindows用メモアプリです。
Markdownエディタ「Obsidian」との連携を前提としています。

## 特徴
- 起動して即メモ開始
- 短い文章でメモを書く
- 複数のメモをまとめてノートにする
- メモの並び替えはドラッグ&ドロップ操作で直感的
- ノートをObsidianのVaultにエクスポート
- メモ単体をObsidianのデイリーノートに追記
- メモデータはローカルに見える形で保存

## 使い方

### 画面の説明
アプリを起動すると、メイン画面である「ボード」が表示されます。
ボードの中央には「メモ」が縦一列に並び、ドラッグ&ドロップで入れ替えが可能です。
左サイドバーには「ノート」の一覧が表示され、各メモをドラッグ&ドロップすることでノートに入れることができます。
あいまいなメモを「ボード」上に書き散らして、あとからノートにまとめることができます。

### メモの追加
ボード画面右下の「＋」ボタンを押すと、最後尾に新規メモが追加されます。ショートカットキー`Ctrl + Enter`でも追加可能です。既存のメモにフォーカスしているときにショートカットで新規メモを追加すると、そのメモの直下に作ることができます。
新しくメモを作ると新規メモに自動でフォーカスされ、すぐに書き始めることができます。メモはあとからいつでも編集が可能です。
メモは自動保存され、作成時のタイムスタンプが記録されます。タイムスタンプはメモの下部に小さく表示されます。
メモのデータはインストールフォルダ内の`data/memo/`内に、個別のJSONファイルとして保存されます。

### ノートの使い方
左サイドバー内の「＋」ボタンを押すと、新規ノートが追加されます。ショートカットキー`Ctrl + N`でも追加可能です。
ノートの名前を入力して確定すると、ノートが作成されます。ノートの名前をクリックすると、そのノートが選択されボード画面上に表示されます。
ボードからメモをドラッグ&ドロップで移動させたり、ノート間でメモを移動させたりすることが可能です。移動したメモはノートの最後尾に追加されます。また、ノート内で新たなメモを作成することも可能です。
ノートのデータはインストールフォルダ内の`data/note.json`に保存されます。

### メモとノートの削除
メモは個別メニューのゴミ箱アイコンから削除できます。削除されたメモは「ゴミ箱」ノートに保管されます。
ノートの削除は、ノートを開いた状態で右上のメニュー内から削除できます。ノート内にあったメモはすべてボードに戻ります。この動作は設定ファイルで変更可能です。
「ゴミ箱」ノート内にあるノートは、右上のメニュー内の削除ボタンで完全に消去できます。この動作は元に戻すことができません。もし再び必要になったメモがあれば、ボードや他のノートにドラッグ&ドロップで戻すことができます。

### ノートのエクスポート
ノートが完成したら、ひとつのMarkdownファイルにエクスポートできます。あらかじめ設定ファイルで保存先フォルダを指定しておく必要があります。
ノートを開いた状態で右上の「📤」ボタンを押すと、エクスポートダイアログが表示されます。必要に応じてノートのファイル名を変更し（デフォルトではノート名が使われます）、ダイアログ内の「エクスポート」ボタンを押すと、指定したフォルダにMarkdownファイルが保存されます。
デフォルトではエクスポートすると同時にノートを削除し、メモをゴミ箱に送ります。この動作は設定ファイルで変更可能です。

### デイリーノートへ追記
メモの内容をObsidianのデイリーノートへタイムスタンプつきで追記します。あらかじめ設定ファイルで保存先フォルダと、デイリーノートのファイル名形式（例：YYYY-MM-DD.md）を指定しておく必要があります。
メモ右下の「…」ボタンから「デイリーノートへ追記」を選択する、またはメモにフォーカスした状態で`Ctrl + D`を押すと、メモ内容がデイリーノートに追記されます。
メモをデイリーノートに追記すると、デフォルト設定ではI My Memo内のメモは自動的に削除されます。この動作は設定ファイルで変更可能です。

### 設定
設定ファイルは、アプリの初回起動時に自動的に作成されます。設定ファイルは、アプリのインストールフォルダにある`config.toml`です。
設定ファイルは、テキストエディタで編集できます。
設定ファイルの内容は、以下の通りです。
```toml
[trash]
note_delete_to_trash = false

[export]
path = "C:\Users\username\Documents\Obsidian\Vault"
auto_delete = true

[daily_note]
path = "C:\Users\username\Documents\Obsidian\Vault"
filename = "YYYY-MM-DD.md"
auto_delete = true
```
